# #089: Interactive Tax Compliance Wizard by Country

**Category:** [FRONTEND]
**Difficulty:** ● MEDIUM
**Tags:** `tax`, `compliance`, `ui`, `localization`

## Description

Create a wizard-based UI that helps Org Admins configure tax rules based on the country of their employees. This should leverage the backend Tax Calculation API and provide a user-friendly way to handle complex local requirements.

## Acceptance Criteria

- [ ] Map-based or list-based country selector for tax configuration.
- [ ] Dynamic form fields that change based on the selected country (e.g., SSN in US, PAN in India).
- [ ] "Tax Preview" calculator for employees to estimate their take-home pay.
- [ ] Warning system for missing compliance data required by specific jurisdictions.
- [ ] Exportable tax summary reports for local government filing.
