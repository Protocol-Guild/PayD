# #092: Contractor Invoicing & Self-Service Portal

**Category:** [FRONTEND]
**Difficulty:** ● MEDIUM
**Tags:** `contractors`, `invoicing`, `portal`, `self-service`

## Description

Expand the Employee Portal to support "Contractors" who need to submit invoices for approval before being paid. This shifts the workflow from "Company-led" to "Contractor-led" for non-salaried workers.

## Acceptance Criteria

- [ ] "Contractor" role added to the RBAC system.
- [ ] Invoice submission form (Hours, Rate, Description, Attachment) in the portal.
- [ ] Admin "Approval Workflow" UI to review, reject, or approve submitted invoices.
- [ ] Approved invoices automatically queued for the next batch payment.
- [ ] PDF generation for submitted invoices for contractor record-keeping.
